# Investigating-the-Role-of-Anaerobic-Bacteria-in-Aggressive-Prostate-Cancer
This is the repository for all the code that I have used in my PhD thesis - organised by chapter along with any relevant files.

The codes have been organised into four chapters in order of my thesis:
1. 16S Sequencing of Frozen Urine Samples
2. Discovery of additional ABBS bacteria - ABBS 2.0 in the PPCG cohort
3. Investigating the relationship between microbial presence, single base substitution mutations and gene families in the PPCG cohort
4. Investigating the relationship between the urine microbiome and the prostate microbiome.
